6. git checkout master - префрлување на master гранката
7. git checkout -b "bug2" - креирање на гранка "bug2"
8. Тука направив git checkout bug1,па после тоа направив промени во SILab1_1, git status, и на крај направив git add .,и како последно додадов git commit -m "..."
